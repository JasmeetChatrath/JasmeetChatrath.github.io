    • Arrays

    An array is a list of items stored between brackets [ ]. We set this list of items to the title of our array. An example of an array is:
    my_array = ["I", "love", "Ruby", 5]
    You can even have an array within an array which is called a multidimensional array.
    multidimensional _array = [[0, 4, 100],["five", "ten", "three"], ["red","blue","green"]]
    The values in our list are assigned to numbers starting at 0. For example, in our my_array example we would use 1 to call our “love” string. To call that value we would enter the following code:
    my_array [1]
    This is called access by index. To call a value in out multidimensional array we use our index values as well. To call “three” for example we would enter the following code: multidimensional _array[1][2]

    Hash

    A hash is a collection of key-value pairs. They are assigned by using "=>" and by setting it to a title. An example of a hash follows:
    my_info = { "first_name" => "Jasmeet", "last_name" => "Chatrath", "age" => 26 }
    To retrieve data in our hash we use the key value. To print “Chatrath” for example we would enter the following:
    puts my_info["last_name"]

    When to use one vs the other

    Ex: We would want to use an array when we have a list of items such as children’s names
    names = ["Jill", "Liz", "Henry", "Anna"]
    We want to use a hash when collecting a set of key value pairs such as favorite items of each child.
    favorite_items ={ "Jill" => "book", "Liz" => "xbox", "Henry" => "car", "Anna" => "makeup" }
